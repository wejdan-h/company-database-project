# Company Database Management System

## Overview

This project presents the design and implementation of a relational database for managing a company's core operations. The system is designed to organize business data efficiently by modeling real-world entities and their relationships, ensuring data integrity through normalization and relational constraints.

## Features

- Requirements Analysis
- Logical Database Design
- Physical Database Design
- SQL Table Creation
- Sample Data Insertion
- Data Retrieval Queries
- Relational Integrity using Primary and Foreign Keys

## Database Entities

- Company
- Manager
- Department
- Employee
- Product
- Customer
- Order

## Technologies Used

- SQL
- Oracle Database
- Relational Database Design



## Purpose

This project demonstrates the complete database development lifecycle, including requirements analysis, conceptual design, logical modeling, physical implementation, and SQL-based database operations.
